== Description ==

This plugin for Related post. This plugin will display your related post with jcarousel slider. It's build with jquery ui jcarsoul plugin.

== Features ==
* Al cross browser supported
* Responsive plugin
* Post display by category
* Post display with jcarousel slider etc

== Installation ==

1. Upload `bh-related-post` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php bh_related_post(); ?>` in your templates
