Установка

  1. Распакуйте папку с плагином в папку wp-content/plugins/
  2. В Панеле Администратора выберите вкладку "Плагины" и нажмите "Установить" рядом с плагином TESTME - Плагин для создания тестов.

Обновление

  1. Если вы не вносили изменения в файл стилей, то загрузите все файлы в папку с плагином.
  2. Если вы вносили изменения в файл стилей, то загрузите в папку с плагином все файлы, кроме testme_style.css. Затем откройте файл testme_style.css из архива и скопируйте данные, которые идут после строки /* ***** Для версии 1.2 *****/ в свой файл стилей.
  3. В Панеле Администратора выберите вкладку "Плагины" и нажмите "Обновить" рядом с плагином TESTME - Плагин для создания тестов.

Создание теста

  1. В меню "Настройки" выберите вкладку "TESTME - Управление"
  2. Нажмите на ссылку "Добавить тест". Такие параметры, как максимульное количество вопросов, ответов и результатов и тип теста потом уже нельзя будет изменить!
  3. Нажмите "Сохранить".
  4. Теперь можно ввести вопросы, ответы и результаты.

    Если тест типа 123, то в зеленых полях рядом с ответом недо ввести число баллов.

    Если тест типа abc, то в зеленых полях рядом с ответом недо ввести букву (или несколько букв), которые соответствуют этому ответу. Подходят любые буквы (русские, латинские) и цифры.

    Для результата надо ввести как минимум 1 из трех полей: "Заголовок результата", "Картинка", "Описание". Можно все три.

  5. Нажмите "Сохранить". Все пустые поля при сохранении теста сотрутся. Потом их уже нельзя убдет добавить.

Добавление теста в запись блога

  1. Добавьте в запись блога код [TESTME N], где N - номер теста. Этот код написан рядом с тестом в таблице управления тестами.

Всякое разное

  1. Текст, который опубликован в записи блога до кода [TESTME N] будет показываться и до вопросов, и до результатов. Текст, который написан в поле "Описание теста" публикуется только перед списком вопросов.
  2. Так как все лишние поля стираются, то в одном тесте может быть разное количество вариантов ответов к разным вопросам.
  3. Для изменения дизайна вопросов, ответов и результатов надо внести изменения в файл стилей testme_style.css .
  4. Я рекомендую придумывать все вопросы заранее, а потом просто вставлять их в тест. Если процесс добавления нового теста не окончен, а надо куда-то уйти, то надо ввести временные записи (например, поставить везде цифру 1), сохранить тест и внести поправки позже. Иначе плагин посчитает все незаполненные вопросы за лишние.

История плагина

Версия 1.2

Спонсор этой версии - Михаил.

  • Добавлена возможность разрешить проходить тест только зарегистрированным пользователям

Версия 1.1
